# Python Automation Projects
Learning Python through building real automation tools.

## Freelance Rate Calculator

### What it does
Calculates monthly freelance income in USD and EUR, checks affodability for a new Macbook Pro M5 16" (€3500).

### How to use
1. Run the script: `rate_calculator.py`
2. Enter your hourly rate
3. Enter hours worked per week
4. Get monthly income calculation and purchase eligibility 

## Skills demonstrated
- User input and type conversion
- Mathematical calculations
- Conditional logic (if/else)
- String formatting (f-strings)
- Currency conversion

## Example output
```
=== Freelance Earnings Calculator ===
Your hourly rate (USD): $50
Hours worked this week: 30

Weekly: $1500.00
Monthly: $6000.00 (€5520.00)
✅ You qualify for a brand new Macbook Pro M5 16"!
```

## Use case
Built for freelancers pricing their services and planning to upgrade their device(s).
